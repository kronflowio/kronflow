A production-grade, cluster-safe, embeddable job scheduler for JVM.

## Status
🚧 Under active development — V1 coming Q2 2026

## What is Kronflow?
Drop one dependency into your Spring Boot app and get a fully observable,
cluster-safe job scheduler backed by your existing relational database.
Zero additional infrastructure required.
